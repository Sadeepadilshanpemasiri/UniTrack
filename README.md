# UniTrack

Your Smart Academic & Campus Companion

UniTrack is a smart campus app designed to help students manage their academic life efficiently. It offers a personalized timetable, study reminders, and a calendar for upcoming campus events and exams. Students can add courses, track deadlines, calculate GPA, visualize academic progress, and use a lost & found board to keep the campus community connected and organized.

---

## Features

- **Personalized Timetable** — Add/edit courses and view a daily/weekly schedule.
- **Study Reminders & Notifications** — Set reminders for classes, assignments, and exams.
- **Calendar & Events** — Campus events, exam schedules, and important dates.
- **GPA Calculator** — Track and calculate semester/yearly GPA.
- **Progress Visualization** — Charts and analytics for study time and grades.
- **Lost & Found Board** — Post and search lost/found items on campus.
- **Offline Suppor**t — Basic functionality works without network connectivity.
- **Dark Mode** — Comfortable viewing in low-light conditions.


## Tech Stack

- **IDE**: Android Studio — official integrated development environment for building Android apps.  
- **Language**: Kotlin — modern, concise, and safe programming language officially supported by Google for Android app development.  
- **Architecture**: MVVM (Model–View–ViewModel) — ensures better code organization, maintainability, and separation of concerns.  
- **UI Design**: XML Layouts & Jetpack Compose — for creating responsive and modern user interfaces.  
- **Database**: Firebase Firestore / Room Database — for storing and managing user and app data.  
- **Authentication**: Firebase Authentication — for secure user login and registration.  
- **Storage**: Firebase Cloud Storage — for saving user files such as images or documents.  
- **Analytics**: Firebase Analytics — to track user engagement and app performance.  
- **Version Control**: Git & GitHub — for code management and collaboration.  
- **Testing & Build Tools**: JUnit, Espresso, and Gradle — for automated testing and efficient builds.
